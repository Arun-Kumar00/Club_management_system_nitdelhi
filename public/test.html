🔐 Google Sign-In (NextAuth)

Authentication is handled by NextAuth — not a raw API call

How login works in this system:
1. User clicks "Sign in with Google" on /login
2. Google redirects back with the user's email
3. NextAuth checks ClubAccess DB — if email exists → session created with clubCode
4. User is redirected to /dashboard
5. Dashboard reads session.user.clubCode and fetches data

To test auth: Just visit your app at the base URL and click "Sign in with Google". All other APIs below require an active session cookie.
